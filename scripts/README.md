# scripts

Carpeta amb notebooks i scripts d'anàlisi relacionats amb l'ús del clúster.

## Fitxers

- `cluster_usage.ipynb`: notebook per analitzar o documentar l'ús del clúster.

## Flux de treball

1. Obre els notebooks amb l'entorn definit a `environment.yml`.
2. Mantén les dades sensibles fora del repositori.
3. Desa els resultats generats en una carpeta adequada i no sobreescriguis els exemples del manual.
4. Si un script passa a ser una eina de manteniment de la web, mou-lo a `code/`.
