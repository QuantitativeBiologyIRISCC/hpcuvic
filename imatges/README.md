# imatges

Carpeta amb imatges i documents enllaçats des del manual web del clúster HPC.

## Fitxers

- `compute_cluster.png`: esquema visual d'un clúster de càlcul.
- `logo_3linies_uvic_color.jpg`: logotip de la UVic-UCC.
- `presentacio.pdf`: presentació general del clúster de càlcul.
- `URespon_cluster.pdf`: tutorial per obrir incidències o sol·licituds.
- `use_resources_efficient.pdf`: guia per interpretar i millorar l'eficiència dels recursos.
- `vscode_remote_ssh.png`: captura de la connexió remota amb Visual Studio Code.

## Flux de treball

1. Afegeix aquí només recursos visuals o documents que s'hagin d'enllaçar des de la web.
2. Fes servir noms descriptius i sense espais.
3. Enllaça el recurs des d'`index.html` amb una ruta relativa.
4. Comprova al navegador que el fitxer s'obre correctament abans de publicar.
