# usage

Carpeta amb material de càlcul i notebooks d'exemple sobre l'ús del clúster.

## Fitxers

- `cluster_calculation.ipynb`: notebook d'exemple per preparar o revisar càlculs al clúster.

## Flux de treball

1. Crea o activa l'entorn de treball amb `environment.yml`.
2. Obre el notebook i adapta les rutes als teus fitxers d'entrada.
3. Executa els càlculs llargs al clúster mitjançant Slurm, no dins del notebook interactiu.
4. Desa només exemples i resultats petits que siguin útils per al manual.
